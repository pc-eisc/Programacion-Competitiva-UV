# Grupo de Programación Competitiva UV 🏆

Repositorio oficial de material y utilidades del grupo de **Programación Competitiva** de la **Universidad del Valle**.

## 📚 Acerca de Este Repositorio

Este es el espacio colaborativo donde compartimos soluciones, guías, recursos y materiales de estudio para el grupo de Programación Competitiva de la Universidad del Valle. Aquí encontrarás todo lo necesario para mejorar tus habilidades en programación competitiva.

## 📁 Estructura del Repositorio

```bash
Programacion-Competitiva-UV/
├── Resources/
│   ├── Books/              # Libros recomendados
│   └── Links.md            # Enlaces importantes y plataformas
└── Readme.md               # Este archivo
```

## 🎯 Recursos Principales

- **[Resources/Links.md](Resources/Links.md)**: Directorio completo de plataformas, comunidades, libros recomendados y enlaces importantes para el grupo.
- **[Resources/Books](Resources/Books)**: Colección de libros recomendados incluyendo "Competitive Programming" de Halim y "Competitive Programmers Handbook" de Laaksonen.

## 🚀 Plataformas Principales

### Aprendizaje y Guías

- [USACO Guide](https://usaco.guide/) - Herramienta principal de organización del grupo
- [Competitive Programming Wiki](https://cp-algorithms.com/) - Algoritmos y estructuras de datos

### Práctica

- [Codeforces](https://codeforces.com/) - Plataforma principal de concursos
- [LeetCode](https://leetcode.com/) - Problemas para principiantes
- [AtCoder](https://atcoder.jp/) - Concursos semanales y mensuales

## 👥 Comunidades del Grupo

- [USACO Guide Group](https://usaco.guide/groups/join?key=JxXgWAZuTACaWVhS0GB5) - Grupo de estudio colaborativo
- [Grupo de WhatsApp](https://chat.whatsapp.com/Dfqg1ycrsYV7AxiNy31WpX) - Comunicación y anuncios del grupo

## 📖 Libros Recomendados

- **Competitive Programming** - Steven Halim y Felix Halim
- **Competitive Programmers Handbook** - Antti Laaksonen (Libro principal del grupo)
- **Introduction to Algorithms** - Cormen, Leiserson, Rivest

## 🤝 Cómo Participar

1. Explora los [recursos disponibles](Resources/Links.md)
2. Únete a nuestras comunidades (USACO Guide y WhatsApp)
3. Practica en las plataformas recomendadas
4. Comparte tus soluciones y contribuye al repositorio
