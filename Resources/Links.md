# Links Importantes del Grupo de Programación Competitiva

## Plataformas de Información y Recursos

* [Usaco Guide](https://usaco.guide/): Un recurso para aprender programación competitiva y prepararse para las competencias. Esta es la herramienta que utilizamos para organizar nuestro grupo de estudio y compartir recursos, se creara un grupo desde donde podemos ver los avances de cada uno (problemas que se han resuelto, los que se están resolviendo y los que se van a resolver, además de tener acceso a guías y tutoriales para cada tema)
* [Competitive Programming Wiki](https://cp-algorithms.com/): Un recurso completo con algoritmos, estructuras de datos y técnicas de programación competitiva.
* [Libros](/Resources/Books): Algunos libros recomendados incluyen "Competitive Programming" de Steven Halim y Felix Halim, "Introduction to Algorithms" de Cormen, Leiserson, Rivest y el libro principal que usaremos "Competitive Programmers Handbook" de Antti Laaksonen.

## Plataformas de Práctica

* [Codeforces](https://codeforces.com/): Una plataforma de programación competitiva con concursos regulares y un gran número de problemas. Esta sera la plataforma principal de donde saldrán la mayoría de los problemas que resolveremos en el grupo, es una plataforma muy popular entre los competidores y tiene una gran comunidad.
* [AtCoder](https://atcoder.jp/): Una plataforma japonesa de programación competitiva con concursos semanales y mensuales.
* [LeetCode](https://leetcode.com/): Una plataforma popular para practicar problemas de codificación, especialmente útil para entrevistas técnicas. Esta es la que usaremos en un principio para practicar problemas de nivel fácil y medio, es una plataforma muy amigable para principiantes y tiene una gran cantidad de problemas categorizados por tema.

## Repositorios de Código
